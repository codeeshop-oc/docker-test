# Reference Commands

### Open Shell Commands
```bash
sudo docker exec -it ___CONTAINER_ID___ /bin/sh
sudo docker exec -it ___CONTAINER_ID___ /bin/bash
```
### Copy
```bash
sudo docker cp ___CONTAINER_ID___:/ ___LOCAL_COPY_DIRECTORY_PATH___
```
